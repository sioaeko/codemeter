CYD Usage Meter

CYD 보드에 Codex 또는 Claude Code 사용량 미터 펌웨어를 설치합니다. 설정은 보드의 Settings 화면에서 조절하고, BLE 장치명은 CYD Usage Meter입니다.

  • CYD board
  • USB data cable
  • Chrome / Edge
  • HTTPS or localhost

Install firmware

화면 방향을 고른 뒤 Install을 누르세요.

v0.4.0
Portrait 기본 세로 배치
Web Serial을 지원하는 Chrome 또는 Edge에서 열어주세요. Web Serial은 HTTPS 또는 localhost에서만 동작합니다.
Landscape 가로 거치 배치
Web Serial을 지원하는 Chrome 또는 Edge에서 열어주세요. Web Serial은 HTTPS 또는 localhost에서만 동작합니다.

Failed to initialize가 뜨면 BOOT 버튼을 누른 채 Install을 누르고, 포트 선택 뒤 Erasing 또는 Installing이 보이면 BOOT에서 손을 떼세요.

Desktop daemon

사용량 전송

펌웨어 설치 후 데스크톱에서 daemon 하나만 실행하세요. Codex와 Claude Code daemon을 동시에 켜면 같은 화면에 값을 번갈아 씁니다.

macOS - Codex

git clone https://github.com/sioaeko/clawdmeter-cyd.git
cd clawdmeter-cyd
codex login
./install-codex-mac.sh

macOS - Claude Code

git clone https://github.com/sioaeko/clawdmeter-cyd.git
cd clawdmeter-cyd
./install-mac.sh

Windows - Codex

git clone https://github.com/sioaeko/clawdmeter-cyd.git
cd clawdmeter-cyd
codex login
py -3 -m venv daemon\.venv
.\daemon\.venv\Scripts\python.exe -m pip install bleak httpx
.\daemon\.venv\Scripts\python.exe .\daemon\codex_usage_daemon.py

Windows - Claude Code

git clone https://github.com/sioaeko/clawdmeter-cyd.git
cd clawdmeter-cyd
py -3 -m venv daemon\.venv
.\daemon\.venv\Scripts\python.exe -m pip install bleak httpx
.\daemon\.venv\Scripts\python.exe .\daemon\claude_usage_daemon.py
제3자 마스코트, 공식 로고, proprietary font 기반 펌웨어 자산은 포함하지 않습니다. Codex와 Claude Code 표기는 호환 대상 설명이며, 이 프로젝트는 OpenAI 또는 Anthropic과 공식 제휴가 없습니다.
