데몬 설치

펌웨어 설치 후 데스크톱에서 데몬을 실행하면 사용량이 Bluetooth로 전송됩니다.

Codex용과 Claude Code용 중 하나만 설치하세요. 둘을 동시에 켜면 같은 화면에 번갈아 값을 써서 표시가 흔들릴 수 있습니다.

OS
표시 대상

macOS - Codex 사용량

Codex CLI에 로그인한 뒤 installer를 실행합니다. 

git clone https://github.com/sioaeko/codemeter.git
cd codemeter
codex login
./install-codex-mac.sh

로그 tail -F ~/Library/Logs/codex-usage-daemon.out.log

macOS - Claude Code 사용량

Claude Code에 먼저 로그인한 뒤 installer를 실행합니다. 

git clone https://github.com/sioaeko/codemeter.git
cd codemeter
./install-mac.sh

로그 tail -F ~/Library/Logs/claude-usage-daemon.out.log

Windows - Codex 사용량

PowerShell에서 수동으로 실행합니다. 

git clone https://github.com/sioaeko/codemeter.git
cd codemeter
codex login
py -3 -m venv daemon\.venv
.\daemon\.venv\Scripts\python.exe -m pip install bleak httpx
.\daemon\.venv\Scripts\python.exe .\daemon\codex_usage_daemon.py

상시 표시하려면 이 PowerShell 창을 열어두세요.

Windows - Claude Code 사용량

Claude Code에 로그인한 뒤 PowerShell에서 수동으로 실행합니다. 

git clone https://github.com/sioaeko/codemeter.git
cd codemeter
py -3 -m venv daemon\.venv
.\daemon\.venv\Scripts\python.exe -m pip install bleak httpx
.\daemon\.venv\Scripts\python.exe .\daemon\claude_usage_daemon.py

상시 표시하려면 이 PowerShell 창을 열어두세요.
